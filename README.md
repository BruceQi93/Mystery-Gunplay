# Mystery-Gunplay
a network FPS Game

### introduce

这是一款能够局域网联机多人对战的FPS游戏，局域网功能是使用Unity3d引擎的新版网络Unet来实现的，目前功能还不够完善，这是1.0版本，后续会添加一些新颖的功能和玩法，期待2.0,3.0.。。。。
